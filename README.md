# Fake News Detector

Analyze news headlines and articles for misinformation patterns using NLP-based scoring.

## Features
- Detects sensational and emotionally charged language
- Identifies excessive CAPS and punctuation abuse
- Scores news 0-100 (Trustworthy → Suspicious → Likely Fake)
- Provides verification tips and fact-checking resources

## How to Use
1. Paste any news headline or article text
2. Click Analyze Now
3. Get instant credibility score with detailed breakdown

HTML | CSS | JavaScript
